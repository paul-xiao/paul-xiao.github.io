autoDeploy

自动部署

安装jenkins

  • 到jenkins官网[https://jenkins.io/download] 下载 rpm包

  • yum 安装

  • 启动jenkins

  • 修改配置(增加jdk路径)

  • 修改防火墙配置

  • 访问http://10.143.132.60:8080/按照提示操作

  • 系统配置
    • 安装插件
      • Deploy to container
      • Maven Integration
      • SSH slave Plugin
    • 节点配置
      • 配置要部署的服务器ip
    • 全局工具配置
      • Ant
      • JDK

  • 新建任务

    • 构建自由风格的软件项目（参考已有任务）
    • 配置svn地址/用户名/密码
    • 编译触发配置（选填 eg: 定时构建 H 12 * ）
    • 编译环境配置
      • Add timestamps to the Console Output
      • with Ant

    • 编译

      • Invoke Ant
        • Ant Version
        • Target (Build.xml name)
      • 准备build.xml 可有eclipse生成，稍做修改即可

    • 编译后配置

      • Deploy war/ear to a container

        • WAR/EAR files (编译后生成的war包)
        • Context paht (项目的上下文 注：tomcat不能配置context,否则会报错)

        • 准备tomcat 

          + tomcat需要有管理界面
+ 配置tomcat-user.xml

        • Contianers (填写tomcat 远程地址及管理账号/密码 注：第一次部署需要删除原有的war包和项目文件夹，并重新启动，否则部署失败)
